# Juego de Adivinar un Número - Java (Consola)

Este proyecto es una aplicación de consola en Java donde el usuario debe adivinar un número generado aleatoriamente por el programa.  
Permite elegir el rango del número secreto y la cantidad de intentos disponibles.

## Características del proyecto

- Menú principal con opciones para jugar o salir.
- Generación de un número aleatorio dentro de un rango elegido por el usuario.
- Validaciones simples:
  - El máximo debe ser mayor que el mínimo.
  - La cantidad de intentos debe ser mayor a cero.
  - Los números ingresados deben estar dentro del rango permitido.
- Mensajes claros para indicar si el intento es alto, bajo o correcto.
- Uso de:
  - Clases y objetos
  - Atributos y métodos
  - Scanner para entrada por consola
  - Estructuras (`if`, `else`, `switch`, `for`, `while`)
  - Paquetes

## Estructura del proyecto
src/
└─ juego/
├─ Main.java
└─ JuegoAdivinar.java

## Ejemplo de uso

- Elegir número de intentos.
- Establecer rango mínimo y máximo.
- Intentar adivinar el número secreto.
- Recibir pistas si el número es más alto o más bajo.
- Ganar o perder según aciertos dentro de los intentos disponibles.
